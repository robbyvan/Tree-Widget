# Tree-Widget
Simple UI to learn OOP

## Syntax
   `var UIInstance = new TreeNode(nodeConfig, outputId);`<br />
   `@param {Object} nodeConfig`<br />
   `@param {String} outputId` <br />
  

## Example
   `const root = new TreeNode({nodeContent: 'Countries'}, 'tree-panel');`<br />
